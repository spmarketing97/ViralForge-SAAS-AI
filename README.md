# ViralForge — The Ultimate AI Viral Content Engine

**ViralForge** is a minimal, professional AI SaaS tool built for content creators, agencies, and marketers. With a single click, it generates viral hooks, SEO-optimized titles, structured video/post scripts, and complete SEO packages for **6 major platforms** — powered by the world's most advanced AI models.

---

## ✨ Key Features

### 4 Output Types — In One Click
| Output | What You Get |
|--------|-------------|
| **🪝 Hooks** | 5 viral opening hooks crafted to grab attention instantly |
| **📌 Titles** | 7 SEO-optimized title variations with power words and click-through optimization |
| **📝 Script** | Full structured script or post: Hook → Intro → Main Content → CTA |
| **🔍 SEO** | Platform-optimized description + tags + target keywords |

### 🎯 6 Platform Targets
- **▶ YouTube** — Long-form scripts, keyword-heavy SEO
- **♪ TikTok** — Short, punchy hooks, trending-style scripts
- **◎ Instagram** — Reels-focused, hashtag-rich, engagement-driven
- **👤 LinkedIn** — Professional, thought-leadership posts
- **👭 Facebook** — Community-driven, conversational posts
- **✕ X (Twitter)** — Concise, high-impact threads/tweets

### 🧠 Multi-Provider AI Support (BYOK)
Users can bring their own API key (Bring Your Own Key) from 4 major providers:
- **OpenAI** (GPT-4o, GPT-4o mini, GPT-4 Turbo, GPT-3.5)
- **Anthropic Claude** (Claude 4.6 Sonnet/Opus, Claude 3.7 Sonnet, Claude 3.5)
- **Google AI Studio** (Gemini 3.1 Pro, Gemini 2.5 Flash/Pro, Gemini 2.0)
- **OpenRouter** (8+ FREE models including Llama 3.3, DeepSeek R1, Mistral, plus premium models)

### 🎨 Content Customization
- **6 Tone Modes** — Energetic, Educational, Funny, Inspiring, Professional, Casual
- **12 Niche Categories** — Tech/AI, Finance, Health, Cooking, Gaming, and more
- **3 Video Lengths** — Shorts/Reels under 1 min, Medium 5-10 min, Long 15-25 min
- **8 Strict Languages** — English, Spanish, Portuguese, French, German, Italian, Dutch, Japanese

### ⚙️ Technical Features
- **100% client-side** — No backend, no database, no deployment required
- **API key stored locally** — Keys never leave the browser (localStorage)
- **Robust formatting** — Advanced JSON parsing prevents generation errors on long files
- **Zero dependencies** — No npm, no Node.js, no build step — just open `index.html`

## 🚀 Quick Start

1. Open `index.html` in any modern browser.
2. Click **"Connect API"** and enter your API key from OpenAI, OpenRouter, Google, or Anthropic.
3. Type your content topic, select the platform, language, and tone.
4. Click **"✦ Generate Content"**.
5. Browse the 4 output tabs and copy what you need!
